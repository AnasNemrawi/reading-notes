# Open-Source Software (OSS)

> Open-source software (OSS) is computer software with freely accessible source code that can be viewed, modified, and distributed by anyone. It promotes collaboration, transparency, and cost-efficiency and is typically developed by a community of contributors. Examples include the Linux operating system and the Mozilla Firefox web browser.
